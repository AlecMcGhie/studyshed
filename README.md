# Quickstart guide

## Step one: clone and locate
```
git clone https://github.com/AlecMcGhie/studyshed

cd studyshed

```

## Step 2: Run startup command

```
./setup.sh
```

## Step 3: Connect to local host

Type http://127.0.0.1:5000 into your browser
